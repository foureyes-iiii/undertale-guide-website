Basics

Undertale is stylised in 2D so the basic controls are the arrow keys to move around. You will be able to check your stats in game and these will change as your LV increases. You have 2 main stats, ATK and DEF. As most games have, there will be save points in many areas which appear as little yellow stars. To save, press enter and navigate the menu that appears.

Encounters

In an encounter, during your turn, you will enter a menu which will show multiple things. The four basic actions are FIGHT, ACT, ITEM and MERCY. When FIGHT is pressed, select the monster to attack and time your attacks (closer to the centre means higher damage). When ACT is pressed, select the monster. Every monster comes with different actions and sometimes this can have an effect on the battle. When ITEM is pressed, any item that is currently in your inventory can be selected and used. MERCY can be used to run away or spare a monster. However this can only be done when their name is highlighted in yellow. The name can be yellow from the start of the battle or can become yellow after doing certain actions.

Locations

There are 5 main areas within the game: Ruins, Snowdin, Waterfall, Hotland and the Core. Each area has a different theme and a different set of monsters. As in any game the difficulty of the monsters will increase as you go through but you should be fine as the gameplay is designed so that it should not be too hard for you to get through.

Endings

Undertale has 3 different endings depending on playstyle: Pacifist, Neutral and Genocide. These are based on how you fight monsters in the game and, as the names suggest, how many you kill. To avoid as many spoilers as possible I will not go into detail about the endings, but I will give the requirements for each ending. The easiest one to do is the Neutral ending. Just play through th game normally and you should get this ending. The Genocide ending can be done without previous playthroughs and the requirement is to kill all the monsters in a specific area of the game. You must continually try to encounter monsters and eventually the encounter image will change into a smiley face. Also, whenever you save there will be a counter showing how many monsters are left. The route can be aborted at any time by sparing a monster. The Pacifist ending, can only be done after a full playthrough of the game. To do this you must spare every single monster you come across without ever gaining EXP.
